📊 Customer Segmentation using Machine Learning
🚀 Project Overview

This project focuses on Customer Segmentation using clustering techniques. The goal is to group customers based on their behavior, spending patterns, and demographics to help businesses make better marketing decisions.

Dataset
Source: Kaggle
Dataset contains customer details such as:
Age
Income
Spending on different products
Purchase behavior (web/store)

🔧 Data Preprocessing
- Converted date columns to datetime format
- Created new features:
- Age from Year of Birth
- Total Children
- Total Spending
- Customer Since (days)
- Removed unnecessary columns
- Handled missing values

- 🤖 Machine Learning Model
Applied K-Means Clustering to segment customers
Used metrics like:
Inertia

📈 Key Insights
Identified different types of customers based on spending
High-income vs low-income behavior patterns
Customer groups for targeted marketing
